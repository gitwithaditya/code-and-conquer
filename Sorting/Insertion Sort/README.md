# 📥 Insertion Sort

*Build the sorted array one element at a time.*

</div>

---

## 📖 Overview

Insertion Sort works by taking one element at a time and inserting it into its correct position in the already sorted part of the array.

It is efficient for small or nearly sorted datasets.

---

## ⚡ Complexity

| Best | Average | Worst | Space |
|:---:|:---:|:---:|:---:|
| O(n) | O(n²) | O(n²) | O(1) |

**Stable:** ✅ Yes  
**In-place:** ✅ Yes

---

## ✅ Pros

- Simple and easy to implement
- Efficient for nearly sorted arrays
- Stable sorting algorithm

---

## ❌ Cons

- Slow for large datasets
- Inefficient compared to O(n log n) algorithms

---

## 💡 Key Takeaway

> Insertion Sort is a great choice for small or nearly sorted arrays, but for larger datasets, algorithms like Merge Sort or Quick Sort are generally preferred.
