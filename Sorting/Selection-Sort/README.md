# 🎯 Selection Sort

*Select the smallest element and place it in its correct position.*

</div>

---

## 📖 Overview

Selection Sort repeatedly finds the minimum element from the unsorted part of the array and places it at the beginning.

Unlike Bubble Sort, it performs fewer swaps but still requires multiple comparisons.

---

## ⚡ Complexity

| Best | Average | Worst | Space |
|:---:|:---:|:---:|:---:|
| O(n²) | O(n²) | O(n²) | O(1) |

**Stable:** ❌ No  
**In-place:** ✅ Yes

---

## ✅ Pros

- Simple to implement
- Performs fewer swaps
- Works well for small datasets

---

## ❌ Cons

- Inefficient for large datasets
- Not a stable sorting algorithm

---

## 💡 Key Takeaway

> Selection Sort minimizes the number of swaps, but its time complexity remains **O(n²)**, making it suitable mainly for learning purposes.
