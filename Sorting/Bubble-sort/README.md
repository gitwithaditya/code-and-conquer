# 🫧 Bubble Sort

*The simplest sorting algorithm based on adjacent swaps.*

</div>

---

## 📖 Overview

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. After each pass, the largest element reaches its correct position.

---

## ⚡ Complexity

| Best | Average | Worst | Space |
|:---:|:---:|:---:|:---:|
| O(n) | O(n²) | O(n²) | O(1) |

**Stable:** ✅ Yes  
**In-place:** ✅ Yes

---

## ✅ Pros

- Easy to learn
- Simple implementation
- Stable sorting

---

## ❌ Cons

- Inefficient for large datasets
- Not suitable for real-world applications

---

## 💡 Key Takeaway

> Bubble Sort is mainly used for understanding the fundamentals of sorting. In practice, algorithms like Merge Sort and Quick Sort are preferred.
